*Created: 2026-05-14*
*Status: Complete*

> **Art Director Sign-Off (AD-ART-BIBLE)**: CONCERNS (accepted) 2026-05-14 — 美术圣经生产就绪。关注事项：(1) 幼者铅白色在宣纸背景上不可见——需记录为有意设计例外；(2) NPC 精灵帧数和房间变体纹理策略需在技术架构文档中澄清；(3) 水墨着色器仍是整个视觉系统成败的关键因素。

## 1. Visual Identity Statement

**一行视觉规则**：理解褪去墨色；无知保存墨色。

> 世界以墨色呈现。你揭开的每一个真相都永久地从场景中洗去一层墨水，揭示其下隐藏的色彩。到第七夜，无所不知的玩家看到色彩丰富的世界；一无所知的玩家仍停留在单色之中。当你不确定一个视觉选择时，问：这代表玩家发现的事物吗？如果是，赋予它色彩。如果不是，让它保持墨色。

### 原则 1：知识着色

每一项发现将特定的颜色永久带入单色世界。墨黑、灰洗、宣纸白是默认调色板。颜色只通过玩家发现真相来获取——从不用作装饰。

*Design test*: 如果不确定某个元素是否应该是彩色的，问：这种颜色是否对应玩家做出的具体发现？如果是，加上。如果纯粹是装饰性的，保持单色。

*服务支柱*: 支柱 1（每个碎片都是拼图）— 每个发现都增添色彩，视觉风格强制执行"没有信息是浪费的"。支柱 3（连接比线索更有力）— 当两个彩色元素在视觉上相关时，联系本身就变得可见。

### 原则 2：墨色承载时间

墨水在旅馆中的表现如同时间的物理显现。安静时墨色沉淀、边缘清晰、世界可读。压力时——倒计时耗尽、事件触发——墨色流淌、晕染、失去边界。时间压力不是 UI 元素，是视觉现象。

*Design test*: 如果不确定场景应有多稳定，问：此刻的游戏内时间压力是多少？高压力 = 墨色流淌、边缘模糊、细节消融。低压力 = 墨色静止、边缘分明、世界清晰。

*服务支柱*: 支柱 2（时间在低语与咆哮之间交替）— 视觉风格将节奏编码进渲染本身。玩家无需看计时器就能感知压力；墨色就是计时器。

### 原则 3：住客以既定颜色书写

每位住客拥有一种源自中国传统水墨调色板的传统色彩。住客出现在场景中时，他们的颜色就存在——但颜色的强度和深度反映玩家对他们了解的程度。未发现的住客是淡薄的暗示；揭露秘密的住客以深沉饱和的色调存在。

*Design test*: 为住客设计视觉表现时，问：这种颜色是否具有文化意义并对应角色的核心秘密？每种住客颜色必须既代表一个角色，又代表一个真相。

*服务支柱*: 支柱 4（每个住客都有要守护的秘密）— 住客在单色世界中通过独特颜色立即可见，同时颜色的强度编码了秘密的揭露程度。

### 视觉决策树

1. 这个元素是否代表玩家发现的事物？是 → 赋予色彩。否 → 保持墨色。（原则 1）
2. 这个时刻是否有时间压力？是 → 让墨色流淌。否 → 让墨色沉淀。（原则 2）
3. 这个元素是否属于住客？是 → 使用他们既定的颜色，按揭露程度调整饱和度。否 → 返回原则 1。（原则 3）

## 2. Mood & Atmosphere

### 游戏状态情绪目标

#### 探索 (Exploration)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 观察好奇心 — 刺探不为你所知的空间的安静紧张感 |
| **光线特性** | 墨色沉淀，边缘锐利。中性温度，低对比度。像雨后潮湿下午在陌生人的房子里。 |
| **氛围描述词** | 沉浸，静默，触觉，内敛，诱人 |
| **能量水平** | 沉思 |
| **情绪载体** | 墨色层本身 — 未发现区域是密集、边缘清晰的墨色；已发现区域墨色更薄、更透明，露出纸纹。 |

#### NPC 对话 (NPC Dialogue)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 审慎亲近 — 与一位说话有所保留的人交谈时的不适亲密感 |
| **光线特性** | 背景墨色微微退去，NPC 指定颜色渗入周围空间。温度向 NPC 色彩偏移。中度对比度。 |
| **氛围描述词** | 亲近，谨慎，带电，脆弱，单向 |
| **能量水平** | 充电 |
| **情绪载体** | NPC 颜色的饱和光晕 — 随玩家揭露秘密，NPC 从灰色剪影变为有血有肉的存在。 |

#### 线索连接 (Clue Connection)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 心智专注 — 将感觉连接成模式的精神紧张 |
| **光线特性** | 场景完全消失，只留纸纹上的墨色。墨色边缘最锐利、最清晰。纯中性温度。 |
| **氛围描述词** | 精准，安静，推演性，网格状，倾斜 |
| **能量水平** | 沉思 |
| **情绪载体** | 连接线 — 灰色起笔，验证后过渡为指示连接有效性的色彩。错误连接保持灰色并褪去。 |

#### 时间压力 — 低语 (Time Pressure: Whisper)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 蔓延不安 — 当前无需惊慌，但时间正在流逝 |
| **光线特性** | 墨色略微颤抖，边缘失去部分锐度。偏冷温度。雨强度增加一级。对比度保持可读但不稳定。 |
| **氛围描述词** | 蠕动，稀释，细雨，不稳定，嗡嗡作响 |
| **能量水平** | 有序 |
| **情绪载体** | 窗上的雨 — 从稳定节奏变为不规则模式，水渍向下流淌。 |

#### 时间压力 — 咆哮 (Time Pressure: Roar)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 倾覆恐惧 — 世界正在瓦解，行动太慢的感觉 |
| **光线特性** | 墨色剧烈流动。边缘大量出血，细节溶解。强烈偏冷（蓝黑墨色）。高对比度，可视区域缩小。 |
| **氛围描述词** | 倾覆，溶解，狂暴，水浸，幽闭 |
| **能量水平** | 急迫 |
| **情绪载体** | 墨色结构失效 — 环境变成活跃对手。门框滴墨，纹理滑落，NPC 剪影边缘渗色。 |

#### 夜晚结束 / 循环重置 (Night End / Loop Reset)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 忧郁接受 — 伴随短暂失败感的宁静臣服，很快被对知识的认知取代 |
| **光线特性** | 墨色完全占据。屏幕淹没在深黑墨水中 2-3 秒，然后缓慢排去，露出干净的白色宣纸。 |
| **氛围描述词** | 淹没，排去，空白，重置，静止 |
| **能量水平** | 静止 |
| **情绪载体** | 墨色排去露出保留色彩 — 玩家已赚取的颜色在白纸上保持可见。进度可视化。 |

#### 发现时刻 (Discovery Moment)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 启示敬畏 — 洞察扩展理解的突然屏息时刻 |
| **光线特性** | 墨色瞬间从特定元素退去。"墨色瀑布"效果，揭示鲜艳饱和色彩。偏暖温度。高对比度。持续 1-2 秒。 |
| **氛围描述词** | 辐射，清澈，扩展，锚定，屏息 |
| **能量水平** | 充电 |
| **情绪载体** | 色彩溢出 — 新颜色略微超出目标边界 10-20px 然后收回，模拟"色墨在宣纸上沉淀"。 |

#### 主菜单 (Main Menu)

| 属性 | 目标 |
|---|---|
| **主要情绪** | 被控制的敬畏 — 安静而沉重的氛围，让玩家感受到旅馆的份量 |
| **光线特性** | 完全单色。深黑墨水在白色宣纸上。高对比度，无温度偏移。雨可见但无声。墨色完全稳定。 |
| **氛围描述词** | 雕刻般，沉重，庄严，寂静，期待 |
| **能量水平** | 静止 |
| **情绪载体** | 宣纸纹理 — 放大的宣纸纤维图案，旅馆轮廓以极简宽笔触描绘。红色朱砂印章是唯一色彩点缀。 |

### 状态转换指导

每个转换经过短暂的中间阶段。情绪不切换，它转变。

| 过渡 | 持续时间 | 视觉机制 |
|---|---|---|
| 探索 → 低语 | 3-5 秒 | 墨色边缘逐渐失去锐度，雨强度上升 |
| 低语 → 咆哮 | 1-2 秒 | 墨色出血突然增加，全屏"墨色闪光"标志阈值 |
| 咆哮 → 夜晚结束 | 即时 | 墨色完全占据 |
| 夜晚结束 → 探索 | 2-3 秒 | 墨色排去，露出白纸和保留色彩 |
| 探索 → NPC 对话 | 1 秒 | 背景墨色减密度，NPC 颜色绽放 |
| 探索 → 线索连接 | 1 秒 | 场景墨色"拉开"，露出笔记本 |
| 任何状态 → 发现 | 即时 + 1.5 秒 | 发现源处墨色瀑布 + 色彩溢出 |
| 主菜单 → 探索 | 3 秒 | 雨声开始，墨色流动从标题向下蔓延 |

### 情绪禁忌

| 禁忌情绪 | 原因 | 危险状态 |
|---|---|---|
| 恐怖 | 恐惧关闭推理。追求惊悚非恐惧。 | 咆哮，夜晚结束 |
| 挫败感 | 循环重置应是信息性的非惩罚性的。 | 夜晚结束 |
| 压倒性混乱 | 即使咆哮状态也必须可读。 | 咆哮 |
| 可爱/异想天开 | 基调是庄重神秘的。 | 探索，NPC 对话 |
| 无菌/临床 | 即使笔记本也必须是手工制品。 | 线索连接 |
| 陈词滥调恐怖 | 闪烁灯光、血红叠加、扭曲脸。水墨语言非数字恐怖。 | 咆哮，夜晚结束 |

## 3. Shape Language

### 基础前提

形状语言的核心问题：**在宣纸上，这笔触是干扫还是湿染？** 干扫产生硬边——断言存在。湿染产生软边——暗示氛围。每个形状要么在断言，要么在溶解，这个二元性直接映射到知识系统和时间系统。

**主规则**：断言形状（干扫边缘）告诉玩家"这很重要，仔细看"。溶解形状（湿染边缘）告诉玩家"时间正在作用于它，感受压力"。

### 角色剪影哲学

#### 无玩家角色

没有可见的玩家角色。玩家是无形的调查者——存在感通过镜头（轻微漂移、呼吸运动）和笔记本中的痕迹体现。NPC 剪影承担全部角色视觉身份。

#### 缩略图可读性规则

每个 NPC 必须在 **48px 高度**（移动端缩略图）可识别：
- 剪影轮廓在 5 个角色中必须唯一
- 最大形状的**几何类别**必须与其他角色不同（高纵、宽横、顶重、底重、中心紧凑）
- 传统色彩必须以低饱和度出现在剪影中（衣摆、发饰、领口的一个色点）

#### 五种原型形状

| NPC 原型 | 主笔触 | 剪影几何 | 轮廓形状 | 情感阅读 |
|---|---|---|---|---|
| **守门人** (管理者，控制信息) | 侧锋宽笔 — 平、宽、建筑感 | 宽水平质量，肩宽于头 | 矩形 — 平顶、方肩、宽底 | 稳定、权威、不可动摇之物 |
| **倾听者** (共情者，吸收他人秘密) | 渲染湿染 — 柔软、吸收、无边 | 圆润无锐角，边缘渗化 | 椭圆 — 软顶、斜肩、无硬终点 | 接纳、舒适、正在消融之物 |
| **观察者** (多疑，从边缘注视) | 中锋枯笔 — 细、锐、碎裂 | 高纵向、棱角、碎片化边缘 | 三角形 — 尖顶、窄框、锐肘 | 紧张、警觉、即将断裂之物 |
| **醉客** (自我欺骗，淹没真相) | 泼墨 — 不规则、失控、渗化 | 不对称、顶重、似将倾倒 | 倒三角 — 重顶、窄底、偏心 | 不稳、压抑、勉强支撑之物 |
| **幼者** (外表无辜，知道更多) | 工笔细描 — 精准、精致、受控 | 紧凑、居中、 disproportionally detailed | 小圆 — 微小框架、大头、清晰边缘 | 藏于微小之中的精密之物 |

#### 水墨剪影设计方法

1. 纯黑墨剪影 — 无内部细节。不可读则失败。
2. 单一色彩点缀 — 在视觉重心处，NPC 的传统色彩，即使 10% 饱和度也可见。
3. 边缘处理随游戏状态变化：
   - 探索：干扫锐利边缘 — 被审视的物体
   - NPC 对话：面向镜头一侧过渡为湿染 — 前面柔软亲密，背面锐利保守
   - 咆哮：所有边缘溶解，仅色彩点缀保持清晰
   - 发现：边缘一帧内从湿到干，然后色彩涌入

### 环境几何

#### 主导几何：纵向直线 + 有机侵入

三层几何，各以不同墨法绘制：

**第一层 — 结构骨架（中锋正笔）**
- 纵向元素：门框、窗棂、柱子、墙角。场景中最锐利的元素。即使咆哮状态也不溶解。纵横比最低 1:8（高窄）。代表旅馆跨循环的永久结构。

**第二层 — 使用表面（侧锋擦笔）**
- 水平表面：桌面、床框、台面、窗台。粗糙、有纹理。纵横比最低 4:1（宽薄）。咆哮时墨从下缘向下流淌——累积时间的重量向下压。

**第三层 — 氛围填充（渲染淡墨）**
- 墙壁、天花板、远处走廊。湿染大片，可见纸纹渗出。无硬边。边界是渐变不是线条。**发现色彩在此层汇集和扩散**。

#### 房间类型与几何主导层

| 房间类型 | 主导层 | 笔触特征 | 效果 |
|---|---|---|---|
| 走廊/过道 | 第一层（纵向） | 密集纵笔，节奏重复 | 压缩、通道、穿越感 |
| 大堂/公共空间 | 第二层（水平） | 宽擦笔，多表面 | 社交空间，可坐可观 |
| 客房 | 第三层（氛围） | 染色主导，边缘全软 | 亲密、私密、封闭 |
| 守门人书房 | 第一层 + 精细 | 纵向 + 表面精工 | 权威、记录、受控知识 |
| 地下室/隐藏空间 | 第三层（重染） | 深饱和渲染，纸纹被遮 | 深度、隐藏、地表之下 |

#### 画即空间规则

房间永远不像加了墨色滤镜的照片。它们必须是**一幅玩家正在行走的画**：
- 刻意不对称：真正的水墨画是有机平衡，非数学对称
- 可见笔触经济：每个大形状看起来是 1-3 笔自信的笔触
- 纸纹即地面：地面平面始终显示原始宣纸纹理——未涂的纸，一切坐在其上
- 负空间即建筑：空白不是缺失细节——是定义绘画空间边界的未画宣纸

### UI 形状语法

#### 双重语域：笔记本在世界中；HUD 在纸上

**语域 A — 线索笔记本（世界内物品）**
- 全屏打开，替换场景视图
- 宣纸表面，可见纤维纹理
- 线索条目为手写注释，墨密度各异
- 连接线为细湿墨线，端点微微晕开
- 有效连接：线条保持并加深。无效连接：线条褪去，留浅水渍
- 无网格、无列、无对齐导线。条目有机排列，位置承载时间意义
- 边缘柔软撕裂——物理物件，非 UI 面板

**语域 B — HUD 元素（纸面叠加）**
- 印章（印章）美学渲染——朱砂红 (#B22222)，微不完美矩形，浮雕纹理
- 计时器不是数字——是角落的朱砂印章，随时间流逝**缓慢开裂**。低语时出现发丝裂纹，咆哮时碎裂
- 夜晚计数器是左侧的印章柱——过去夜已盖印，当前夜半盖，未来夜空白
- 交互提示以红色墨水小注形式出现在物品旁

#### 形状层级

| 优先级 | 形状 | 墨处理 | 尺寸规则 |
|---|---|---|---|
| 关键 (计时器裂纹、发现通知) | 印章，强边 | 朱砂红，全饱和 | 不小于 32x32 px |
| 标准 (夜晚计数、交互提示) | 印章，标准边 | 朱砂红，60-80% | 16-24 px |
| 氛围 (房间名、状态指示) | 淡墨注释 | 灰墨，20-40% | 10-14 px，3 秒后淡出 |

### 主次形状与视觉层级

#### 墨色经济规则

每个场景的墨色预算：
- 1-2 个主要元素（密墨、锐边）— 线索物品、NPC
- 4-6 个支撑元素（中等墨、纹理边）— 家具、建筑
- 无限退隐元素（轻染、软边）— 墙壁、天花板、远处空间

超过 3 个主要元素 = 过载。零个 = 无关紧要。

#### 通过对比引导视线，非箭头

游戏永不使用视觉箭头、高亮、光效或地图标记：
1. **密度对比**：视线趋向最密墨块
2. **边缘对比**：在柔边世界中，单一锐边拉住焦点
3. **色彩对比**：单色世界中任何颜色都是灯塔
4. **负空间框架**：主要物体周围留更多空白

#### 咆哮时层级降级

- 建筑纵线保持锐利（空间锚点）
- 支撑物体开始失边
- 主要物体（线索）保持锐度但墨色开始流淌
- NPC 压缩为纯剪影 + 色彩点缀

#### 温色转换规则

1. 整个游戏在夜间进行。温色是空间性的，非时间性的。

4. **循环重置恢复** — 每夜开始温色回基础值。

### 色盲安全分析

| 色对 | 受影响缺陷 | 备份机制 |
|------|----------|---------|
| 靛蓝 vs 梅紫 | 红绿色盲 (~8%) | 剪影形状不同（三角 vs 倒三角）+ 笔记本卡片切口模式 |
| 青瓷 vs 藤黄 | 蓝黄色盲 (~0.01%) | 连接线脉冲动画（2s 呼吸）vs 青瓷静态平滑 + 图标区分 |
| 赭石 vs 朱砂红 | 红色盲 (~1%) | 上下文分离：朱砂仅 UI 叠层位置，赭石仅游戏世界内 |
| 温色偏冷 | 全色盲（极罕见） | 墨色边缘退化——形状语言变化是主要时间压力信号 |
| 大堂五色共存 | 任何严重缺陷 | 空间分离 + 独特剪影 + 色点 |

#### 通用备份规则

1. 永不单独依赖色彩。每种语义色配以形状、位置、图案或动画提示。
2. 笔记本是安全回退。所有色码线索在笔记本中也有文字标签。
3. 音频强化发现色彩。每种住客色有独特乐器音色：靛蓝=古琴，赭石=木鱼，青瓷=玻璃碗，梅紫=二胡，藤黄=颂钵。

### 饱和度公式

住客色饱和度：`guest_saturation = 10% + ((discoveries / total_secrets) * 90%)`

藤黄饱和度：`gold_ochre = 40% + ((connections_made / total_connections) * 60%)`

朱砂红恒定 100%。

## 5. Character Design Direction

### 视觉原型

所有 NPC 取自明清人物画传统，非动漫或西方游戏风格。参考文人画——形象简括、笔触传神、心理密度高。

**观察者**（靛蓝 #2E4A7A | 三角形 | 中锋枯笔）
- 时代：晚清落魄文人。瘦削、棱角、营养不良。
- 服饰：过大的旧长衫，袖口过长半遮双手。曾经的考究已消退。
- 年龄：50-60s，消瘦使年龄难辨。
- 姿态：永远倾斜——靠墙、靠门框、靠家具。从不居中。三角来自突出的锐利肘部和警觉的头部倾斜。
- 识别特征：过长的袖子，微微抬起但从不完成手势——"枯笔"的碎片感。

**守门人**（赭石 #8B4513 | 矩形 | 侧锋宽笔）
- 时代：民国旅馆经理——以现代自居但权威来自旧传统。1930s 上海旅业者移居山间。
- 服饰：中山装外套配传统裤。宽阔肩线、方正剪裁、填充水平空间。腰间一串铜钥匙环——唯一的非墨非色元素（纯白负空间）。
- 年龄：45-55，敦实，保养良好。
- 姿态：正面、居中、重心均匀。双手背在身后或单手平按表面——宣示领地。
- 识别特征：腰间白色钥匙环。48px 缩略图中是矩形中心的白色圆点。唯一正面朝向镜头的 NPC。

**倾听者**（青瓷 #7BA691 | 椭圆形 | 渲染湿染）
- 时代：超时代——任何世纪皆可。民间女性，视觉锚点不在服装年代而在材质品质。
- 服饰：宽松早期旗袍 + 编织披肩。衣料垂坠流淌，无可见硬缝。披肩边缘以湿染融入背景。
- 年龄：30-40s，圆润使年龄偏年轻。
- 姿态：微驼、肩内收、双臂贴身。保护性、吸收性。椭圆来自披肩的连续曲线。
- 识别特征：披肩是唯一始终以湿染边缘渗入背景的衣物——轮廓永远部分不确定，"吸收"环境。

**醉客**（梅紫 #6B3A5E | 倒三角 | 泼墨）
- 时代：1920-30s 失意知识分子——借酒浇愁的诗人或记者。西化但传统教育。
- 服饰：西装外套敞开配皱巴的传统内衫。文化冲突是核心——介于两世界、两不归属。外套创造顶部重量，细腿和倾斜创造窄底。
- 年龄：35-45，饮酒使年龄不一致——脸浮肿、手瘦削。
- 姿态：不对称倾斜——永远偏离垂直 10-15°，重心在单肘或单髋。倒三角来自宽夹克下的窄站姿。
- 识别特征：内袋露出的酒瓶颈——缩略图中是打破剪影的对角线，唯一带突出物的 NPC。

**幼者**（铅白 #E8E0D4 | 小圆 | 工笔细描）
- 时代：刻意模糊——不固定于任何时期。幼者的叙事性质本质不明。
- 服饰：简单对襟短衣和裤子。无年代标记。但以工笔风格渲染——不可思议的精细细节（纽扣针脚、领口精致纹样），其他 NPC 在此尺度无此细节。
- 年龄：8-10 岁，比年龄小。不成比例的大头创造"小圆"形状。
- 姿态：紧凑、静止、双手握于身前或握一物。异常沉着——静止本身就是信号。
- 识别特征：工笔细线本身。其他 NPC 大笔触渲染，幼者可见地不同——更精细的线条技术。

### 即时识别三层系统

| 层级 | 机制 | 实例 |
|------|------|------|
| **剪影几何** | 矩形/椭圆/三角/倒三角/小圆 | 48px 可读 |
| **独特配件** | 钥匙环/披肩/袖子/酒瓶/精细细节 | 一目了然 |
| **朝向** | 正面/3/4前/3/4后/侧面/低头 | 不重叠 |

### 表情/姿态风格

**克制水墨自然主义。** 情感通过姿态和笔触质量传达，非面部表情。

面部规则：两个点（眼睛角度=情感——上扬=警觉，下垂=悲伤，水平=中性）+ 一笔口。无鼻。面部在 720p 约 15-20px 高。

情感通过笔触质量表现：
- 防御/撒谎：边缘锐化（干扫增加），色彩退缩
- 敞开心扉：边缘柔化（湿染增加），色彩温暖扩展
- 焦虑/压力：纵向下滴线出现，形态开始不稳定
- 被抓到谎言：横跨中段的锐利水平裂痕 0.5 秒——仅观察者注意到

姿态变化：每个 NPC 最多 2-3 个姿态（默认/对话/焦虑）。过渡非动画而是通过墨色溶解交叉淡入。

### LOD 哲学

固定摄像机 1280x720，无缩放。NPC 渲染高度约 200-280px。

- **保留**：剪影几何、独特配件、色彩点缀、笔触边缘质量、整体姿态
- **最小保留**：面部特征（2 点眼、1 笔口）、领口/衣边细节、手位
- **完全省略**：织物纹理、手指细节、耳鼻细节、发丝

48px 缩略图不可读的细节不存在。每笔服务于识别或叙事。

## 6. Environment Design Language

### 建筑风格

**文化锚点**：民国山间旅馆，约 1930-40s。中西混合——建筑处于两个身份之间，映射游戏中表面秩序下的隐藏真相。

**建筑语汇**：
- **结构**：木框架，走廊和公共空间可见梁。地下室可见石基。无钢无混凝土——有机材料会老化、染色、呼吸。
- **墙壁**：白灰泥覆砖，角落和地下室局部磨损露出底层。客房木护墙板到椅背高度。
- **地面**：宽板深木（走廊、大堂、餐厅）、石砖（地下室）、窄席（客房）。地面始终是最原始材质——最接近纸。
- **窗户**：木框多格。内百叶而非窗帘。玻璃上的雨是主要氛围元素。
- **门**：镶板木配铜五金。沉重。门框始终是第一层元素——永不溶解。
- **楼梯**：中央建筑锚点。锻铁栏杆配木扶手。

**不引用**：无斗拱、无朱柱、无皇家尺度。这是朴素商业建筑，非寺庙。无装饰艺术几何装饰——西方影响是结构性的（门比例、五金、房间布局），非装饰性的。

### 纹理哲学

**核心决定：全手绘纹理，零 PBR。**

理由：
1. 水墨着色器操作 canvas_item——转换像素数据。着色器期望绘画，非照片写实。PBR 贴图（法线、粗糙度、金属度）无可用通道。
2. "画即空间"原则要求每表面在像素级读作笔触。
3. 性能：手绘纹理更小、压缩更好、渲染遍数更少。100 draw call 预算下的生产约束。

**各层纹理规格**：

| 层 | 来源 | 分辨率 | 着色器交互 |
|---|------|--------|-----------|
| 第一层（结构） | 手绘干笔 | 512x512 | 边缘像素必须干净二值 alpha |
| 第二层（使用面） | 手绘擦笔 | 512x512 | 底部 16px 密度更高以种子流淌效果 |
| 第三层（氛围） | 着色器程序生成 | 艺术家提供 256x256 密度遮罩 | 着色器完全生成此层 |

**纹理命名**：`env_[room]_[layer]_[element]_[variant].png`

**宣纸纹理**：共享 1024x1024 可平铺宣纸纹理。从真实宣纸高分辨率扫描清洁至无缝。作为所有房间的地面平面。第三层渲染时以 15-25% 透明度可见。

**纹理图集**：每房间一个 2048x2048 图集。包含该房间所有第一层和第二层纹理。1280x720 基准分辨率下单图集保持低 draw call。

### 道具密度规则

**核心原则**：每个可见物品要么是故事线索，要么是欺骗。没有装饰。

| 房间类型 | 交互道具 | 非交互道具 | 总计 |
|---------|---------|----------|------|
| 走廊 | 0-1 | 2-4 | 3-5 |
| 大堂 | 2-3 | 6-8 | 8-11 |
| 餐厅 | 2-3 | 8-10 | 10-13 |
| 客房（各） | 3-5 | 4-6 | 7-11 |
| 书房 | 4-6 | 6-8 | 10-14 |
| 地下室 | 2-3 | 3-5 | 5-8 |
| 阁楼 | 1-2 | 2-4 | 3-6 |

**密度规则**：
1. **80/20 规则**：80% 非交互（环境语境），20% 交互（线索承载）。交互前不可视觉区分——相同墨色渲染，hover/时才出朱砂提示。
2. **剪影测试**：每个道具必须有独特黑色剪影。两件共享剪影的必须移除或重塑。
3. **NPC 房间讲述主人故事**：每间客房含 1-2 件潜意识编码住客秘密的道具——首次不理解，重玩时才显而易见。使用 NPC 色彩 10-15% 饱和度。
4. **循环变体密度**：跨夜变化的道具不需要新纹理——相同资产重定位、旋转或墨密度调整。书从桌到地是同一 sprite 不同位置。
5. **每房间最多 15 件独特道具**。超出则使用共享道具库（书、瓶、布品跨房间共享）。
6. **欺骗道具**：每房间恰好 1 件看起来像线索但不是的道具。比其他道具略锐的墨边（第一层处理）吸引目光。每夜更换。

### 环境叙事指南

**原则：物件是证词。每个房间是证人陈述。**

三个平行故事：
1. **旅馆的故事** — 建筑见证了什么。通过磨损、污渍、修补、结构变化讲述。每间可见。
2. **NPC 的故事** — 住客隐藏了什么。通过有意放置的个人物品讲述。局限于住客房。
3. **循环的故事** — 自昨夜什么改变了。通过位移、缺失、新痕迹讲述。分布在所有房间。

**NPC 秘密物件编码三层**：

| 层 | 目的 | 可见性 | 循环持久性 |
|---|------|--------|----------|
| **表面布置** | NPC 想让你看到的 | 立即可见 | 每夜重置 |
| **隐藏物件** | NPC 主动隐藏的 | 部分遮挡——后面、下面、里面 | 重置，但隐藏位置持久（抽屉总有假底） |
| **磨损证据** | NPC 无法隐藏的 | 细微——家具底面污渍、地板磨损路径 | 跨循环持久（永久结构证据） |

**磨损即真相**：磨损图案以第一层锐度渲染，即使出现在第二层或第三层表面。磨损路径以硬干笔绘制，而周围地板是柔软渲染。这让磨损在视觉上"断言"——画作告诉玩家这笔触重要。

**缺失即线索**：餐桌少一把椅子（地板磨损不同可见），墙上空钉孔（曾挂之物），衣帽钩上的拉伸痕迹（重衣被暴力拉下）。

**地下室是汇聚点**：所有 NPC 的磨损证据汇聚于此。多来源污渍重叠。道具少但信息密度高。五个独立房间的线索在这里变成一个故事。

**阁楼是元层**：旅馆前时代的储存物品。不关于当前 NPC，关于建筑本身作为角色。阁楼道具使用零住客色——纯单色，即使发现后也真正无色。

**咆哮状态叙事**：墨色溶解优先：
1. 溶解当前房间的隐藏层（使隐藏物品更难发现）
2. 保留磨损证据（硬边保持硬——真相抵抗溶解）
3. 从隐藏位置向外渗出 NPC 色彩（秘密在压力下泄漏）

观察者学会阅读渗出方向——咆哮使房间更难导航但微妙揭示秘密位置。

## 7. UI/HUD Visual Direction

### HUD vs 叙事 UI 分离规则

| 语域 | 本质 | 例子 | 字体 |
|------|------|------|------|
| **语域 A — 世界内物品** | 玩家在游戏世界中发现和使用的物理物件 | 笔记本、线索卡、连接线 | 楷体（KaiTi）— 手写感 |
| **语域 B — 系统叠加** | 游戏对玩家说话的声音 | 计时器印章、夜晚计数、交互提示 | 仿宋（FangSong）— 官方印刷感 |

规则：楷体 = "世界内的知识"，仿宋 = "游戏对你说话"。任何一眼可见的字体差异立即区分信息来源。

### 字体层级

| 级别 | 字体 | 大小 | 用途 |
|------|------|------|------|
| H1 | 楷体 Bold | 24-28px | 笔记本标题、发现通知标题 |
| H2 | 仿宋 Regular | 18-22px | HUD 印章文字、菜单选项 |
| Body | 楷体 Regular | 14-16px | 笔记本条目、对话文字 |
| Caption | 仿宋 Light | 10-14px | 房间名、氛围文字（3s 淡出） |

### 图标风格

**篆刻印章风格。** 每个 HUD 图标是一个微印章——红色小矩形内含简化篆字或几何图形。与语域 B 的印章系统一致。笔记本不使用图标——使用手写文字。

### UI 元素清单

#### 计时器（语域 B）
- 位置：右上角
- 形式：朱砂印章，随时间流逝开裂
- **UX 修复**：点击/长按印章显示墨色倒计时数字（3s 淡出）。低语时发丝裂纹，咆哮时碎裂。每 5min/2min/1min/30s 音频提示。
- 空闲时微呼吸动画（2-3px 振荡，4s 周期）保持边缘感知

#### 夜晚计数器（语域 B）
- 位置：左上角
- 形式：纵向印章柱——过去夜已盖印（100%），当前夜半盖（60%），未来夜空白
- 间距向下递增——时间加速感

#### 交互提示（语域 B）
- 形式：朱砂墨水小注，手写风格，出现在物品旁
- 触发：鼠标 hover 0.5s 后出现（PC），或 3s 淡出
- **移动端**：交互物品每 8-10s 轮流闪一次"湿光泽"（非高亮，单帧光泽），替代 hover 发现

#### 对话 UI（新定义）
- 对话是独立游戏状态，非 HUD 叠加
- 场景变暗（墨色退去），底部 30% 成为宣纸对话面板
- NPC 文字用其住客色渲染
- 玩家选项为朱砂印章按钮
- 计时器继续但降速至 50%
- 可在对话中打开笔记本（交叉参考）
- 明确的"结束对话"选项始终可见

#### 线索笔记本（语域 A）
- 全屏宣纸，可见纤维纹理
- 楷体手写条目，墨密度各异
- 连接线：细湿墨笔触，端点微晕。有效=藤黄，无效=灰色褪去
- **多页物理笔记本**：每页 8-12 条线索，滑动/滚轮翻页
- **搜索**：键入时匹配条目墨色微亮（"湿手指描"），无搜索栏
- **移动端**：点击选择 + 点击连接（替代拖拽）+ 捏合缩放至 2.5x
- 跨循环条目：旧条目"较干"（低饱和、稳定边缘），新条目"较湿"（更深、软边）
- 咆哮时笔记本免受世界墨色溶解——知识抵抗消融

### 动画感觉

| 过渡 | 持续时间 | 效果 |
|------|---------|------|
| HUD 元素出现 | 0.3s | 印章"盖下"——从略大缩至实际尺寸，微弹跳 |
| HUD 元素消失 | 0.5s | 墨色淡出（非 alpha 淡出——墨密度降低） |
| 笔记本打开 | 0.5s | 场景如翻书页般关闭，笔记本翻开 |
| 笔记本关闭 | 0.3s | 纸面折回场景 |
| 对话进入 | 1.0s | 场景墨色退去，NPC 色绽放，纸面从底部升起 |
| 发现通知 | 0.5s 出现 + 2.0s 保持 + 0.5s 褪去 | 金色藤黄边框环绕，如墨滴入水 |

### 无障碍

- **对比度**：HUD 元素在深墨底上加半透明宣纸背板。朱砂红 `#B22222` 对宣纸白 `#E8E0D4` 约 4.3:1（临线），背板确保实际可读性
- **高对比度模式**：HUD 切换为白墨+半透明深背板——时代适当
- **音频时间提示**：5min/2min/1min/30s 剩余时单声软鸣（墨溅音效）
- **色盲**：每种语义色配有形状/位置/图案/动画备份（见 Section 4）

### 移动端适配

- 19.5:9 宽高比：HUD 锚定顶部安全区，底部完全留空供触控
- 交互方式：tap 替代 click，长按替代右键，pinch-to-zoom 笔记本
- 无 hover-only 交互——湿光泽循环提供被动可发现性
- 对话面板触控友好：选项最小 44px 高度

## 8. Asset Standards

### 文件格式

| 格式 | 用途 | 色彩空间 |
|------|------|---------|
| **PNG-24 (RGBA)** | 环境纹理、角色贴图、图集 | sRGB |
| **PNG-16 (灰度)** | 墨色密度遮罩、着色器查找纹理 | Linear |
| **SVG** | UI 印章图标 | sRGB |
| **OGG Vorbis** | 音乐、环境循环 | — |
| **WAV 16-bit PCM** | UI 音效、发现提示音 | — |
| **GDShader** | 所有视觉效果 | — |

**禁止格式**：JPEG（损失压缩破坏墨边）、BMP（无 alpha）、WebP（Godot 导入不稳定）

### 纹理分辨率层级

| 层级 | 分辨率 | 资产类别 |
|------|--------|---------|
| T1 图集 | 2048x2048 | 每房间一个（8 房间） |
| T2 标准 | 512x512 | L1/L2 单独元素 |
| T3 角色 | 256x512 | NPC 精灵表 |
| T4 细节 | 256x256 | L3 密度遮罩、小道具 |
| T5 UI | 64-128px | HUD 印章图标 |
| T6 可平铺 | 1024x1024 | 宣纸纹理 |

**绝对上限**：无纹理超过 2048x2048。移动端兼容硬约束。

### Godot 4.6 导入设置

**环境和角色纹理**：
- 压缩：VRAM Compressed（桌面 s3tc，移动 etc2/astc）
- Mipmap：图集启用，UI 禁用
- Alpha：Premultiply 启用，Fix Alpha Border 启用
- 色彩空间：sRGB
- 移动端覆盖：`import_etc2_astc = true`

**密度遮罩**：Lossless（保留梯度精度）

**宣纸纹理**：VRAM Compressed + Mirror Repeat（隐藏平铺接缝）

**音频**：OGG 循环启用（环境/音乐），WAV 零延迟（SFX）

### 内存与性能预算

**每房间热内存**：~6.4 MB（图集 ~2.7 + 宣纸 ~0.7 + NPC ~1.0 + 遮罩 ~0.03 + UI ~0.2 + 着色器 ~1.8）

**Draw call 预算**：每房间 15 个硬上限（总计 100 帧预算）

**着色器 GPU 时间**：
- 墨色渲染着色器：1.5ms max
- 雨着色器：1.5ms max
- 后处理总计：3.0ms max
- 着色器指令：每片段 <64 ALU，<4 纹理采样

## 9. Reference Direction

### 参考 1：黄宾虹（1865-1955）— 晚年山水

**提取**：「五笔七墨」体系——浓墨、破墨、泼墨在单幅构图中的分层运用。极端密度区与近空区的过渡从不平滑，而是边缘碰撞。映射到三层环境系统（L1 结构/L2 表面/L3 氛围）和墨色经济规则。建筑形式（桥、亭、路）以墨笔触保持几何，即使周围混沌——对应"第一层永不溶解"规则。

**避免**：黄宾虹晚期作品的压倒性密度。本游戏默认状态是稀疏、安静、呼吸的负空间。密度仅用于特定房间（地下室汇聚点）或特定状态（咆哮）。

### 参考 2：Okami（Clover Studio, 2006）— 天笔与墨色填充

**提取**：渲染器如何在几何体上保持墨线边缘，同时内部填充渐变色洗——L1/L2/L3 分离的技术模型。「修复」机制中色彩回流进入褪色区域的转场时间：约 0.5s 退去 + 0.8s 色彩升起 + 0.3s 溢出-稳定。直接映射到发现时刻状态。

**避免**：Okami 的温暖饱和色调和日式（浮世绘 + 水墨混合）风格。本游戏是冷调中性 + 中国传统颜料。Okami 的世界是欢快神话的；本游戏是紧张心理的。仅参考墨色渲染架构和修复转场时间。

### 参考 3：Gorogoa（Jason Roberts, 2017）— 面板即窗户构图

**提取**：每个面板既是完整插画，也是建筑视角——「画即空间」规则的技术模型。不对称构图以重量平衡而非镜像，墨密度决定叙事焦点——「通过对比引导视线，非箭头」原则。

**避免**：Gorogoa 的温暖水彩插画风格和西方绘本质感。本游戏的视觉语言必须读作东亚水墨画。仅参考构图纪律：框架、不对称、焦点放置。

### 参考 4：梁楷（13 世纪）— 减笔人物画

**提取**：《李白吟行图》和《泼墨仙人图》作为 NPC 角色渲染的主要参考。每个角色仅 8-15 笔定义，每笔承载个性信息——48px 缩略图 LOD 哲学的模型。通过姿态、衣褶和笔触能量区分个体，非面部细节。同一人物服饰内枯笔与泼墨共存——NPC 边缘状态转换的视觉模型。

**避免**：梁楷的禅宗哲学框架——本游戏角色是心理复杂、困顿和欺骗性的。笔触经济可借鉴，禅意空灵不可。梁楷人物存在于空白空间，本游戏环境遵循三层系统。

### 参考 5：王家卫 —《花样年华》（2000）— 封闭空间电影摄影

**提取**：约束性居住空间同时感觉亲密和幽闭的视觉语法。走廊框架中纵线压缩画面，视线被吸引到唯一光亮区域——走廊「第一层主导」的构图模型。纺织品（窗帘、壁纸）作为中性环境中的情感色彩源——NPC 色彩"渗入"房间的模型。同一空间因情感语境而呈现不同色彩温度——房间温色系统的模型。

**避免**：王家卫的浪漫忧伤基调和饱和红绿上海内景。本游戏不是爱情片——张力来自调查而非情欲。朱砂红信号权威和系统反馈，非热情。仅参考空间语法：走廊压缩、织物即色源、建筑内的情感温度转换。
